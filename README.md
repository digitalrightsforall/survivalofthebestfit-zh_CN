## 关于本项目

这是 **Survival of the Best Fit** 的中文翻译版本，由「普通人的数字权利」出品。

### 普通人的数字权利简介

「普通人的数字权利」是一个面向普通市民开展提升全民科技伦理素养，提升全民数字权利意识，为普通人提供数字权益发声和参与平台的公益社区，我们试图通过互动工作坊、线上资讯传播、知识出版等方式来赋能每一个普通人，让普通人提升科技伦理素养，建立数字权利的意识，掌握必要的法定权利和工具，并共同探讨未来参与式治理科技并主张数字权利的新方式。

更多详情可访问：https://digitalrights.opendatachina.org

## 游戏概念

适者生存是一个教育性网络模拟游戏，旨在揭示 AI 和 ML 算法如何可能存在偏见，特别是通过研究在招聘实践中使用 ML 如何加剧劳动力市场不平等。它旨在打破互联网用户与构建互联网技术（特别是 AI 和 ML）之间的壁垒，使他们了解招聘应用程序开发者的决策如何直接影响算法产生的决策。

## 目标和潜在影响

「因此，只要我们仅仅设想和推进技术、忍受它或逃避它，我们就永远不会体验到我们与技术本质的关系。」——马丁·海德格尔

作为年轻的技术爱好者，我们认为我们有责任帮助人们理解算法及其对社会的影响。在研究这个主题之后，我们发现算法招聘是一个有问题的领域，因为招聘实践的保密性和高度自动化。我们的目标是通过研究和用户测试找到提高公众意识的方法。在这样做之后，我们得出结论，一系列基于网络的游戏化模拟可以向数千名具有任何技术知识水平的访问者介绍招聘中算法的问题。
